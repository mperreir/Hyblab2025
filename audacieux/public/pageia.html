Les Plus de l'IA

  • Base de données solide permettant un travail collaboratif sans IA
  • Illustrations vectorielles efficaces pour le développement web
  • Permet d'être plus rapide
  • Permet une cohérence de A à Z
  • Rapidité dans la génération d'images

Les Limites de l'IA

  • Peu efficace pour le style et la mise en page
  • Perd rapidement le contexte
  • Génère des illustrations imparfaites
  • Besoin de beaucoup de prompts
